# c1lgkt-jax/bin

This folder contains notebooks and python scripts that use the c1lgkt library to perform various tasks and analyses.
The layout is something like:

* `dev/`: Notebooks/scripts used to help develop the library
* `scratch/`: Scripts which aren't really part of the library, more for general learning

In addition, here are some physics scripts:
* `negative_triangularity/`: Scripts involved with analysis of negative triangularity discharges
* `rmp/`: Project w/ resonant magnetic perturbations
* `nontwist/`: Purely physics; analysis scripts for nontwist tori
